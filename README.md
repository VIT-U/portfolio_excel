# Портфолио проектов по анализу данных с использованием Excel.
> Данный репозиторий содержит проект, выполненный в рамках практической части курса по анализу данных. Основной фокус — применение **Excel** для решения бизнес-задач, анализ метрик и формирование выводов.

---

## Краткое описание
* Провел исследование продаж по магазинам и категориям товаров.
* Изучил платежеспособность по странам и количество клиентов в этих странах с использованием математических функций.
* Составил дополнительные сводные таблицы.
* Провел работу по исправлению данных в именах клиентов и привел их к стандартному виду с использованием текстовых функций.
* Составил портрет наиболее лояльных клиентов.
* Создал единый дашборд.

> С работой можно ознакомится на [Google диске](https://docs.google.com/spreadsheets/d/1VtKr9KBqCe8h5KaSqFHHic9UT55Y3ZpI/edit?usp=sharing&ouid=101142551206336549858&rtpof=true&sd=true), но полный функционал будет отражен в Excel приложении.
